# Scripts

This folder contains the main runnable scripts of the project.

## Files

- `train_task4_peak_mlef.py`  
  Main training and evaluation script for the Task-4 peak forecasting setting with MambaFormer-TPEF and MLEF ensemble.

- `generate_paper_figures.py`  
  Script for generating figures used in the paper and project documentation.

- `shap_analysis_task4.py`  
  Script for SHAP-based interpretability analysis for Task-4 experiments.

## Notes

- Core model components, training utilities, and data processing modules are implemented under `src/`.
- The scripts in this folder mainly serve as entry points for experiments and analysis.
