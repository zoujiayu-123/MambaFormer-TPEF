# Sample Data

This folder contains a small subset of sample CSV files used only to illustrate the input data format.

## Purpose
The sample files are provided for:
- demonstrating the expected data structure,
- showing example column organization,
- helping users understand how to prepare their own data.

## Notes
- These files are **not** the complete dataset used in the experiments.
- Users should download the full public dataset from the official source if full reproduction is needed.
