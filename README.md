# MambaFormer-TPEF
Official code for "MambaFormer-TPEF: A Hybrid State Space-Transformer Framework with Two-stage Peak Enhancement for Ultra-long Sequence Photovoltaic Power Forecasting".

## Scripts

The main runnable scripts are located in the `scripts/` directory:

- `train_task4_peak_mlef.py`: main training and evaluation script for the proposed framework.
- `generate_paper_figures.py`: script for generating figures used in the paper.
- `shap_analysis_task4.py`: script for SHAP-based interpretability analysis.
